# COMAP 2023 MCM: Predicting Wordle Results

## Setup

1. Install **uv** (if not already installed):

```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. Install project dependencies:

```sh
uv sync
```

## Notebooks

Run the notebooks in the following order:

1. `data_cleaning_and_exploration.ipynb` — initial data analysis and preprocessing
2. `1.ipynb` — feature engineering and baseline modeling
3. `2.ipynb` — final models and evaluation
